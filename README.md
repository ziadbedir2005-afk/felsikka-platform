# Felsikka Ride-Sharing Platform

A conceptual ride-sharing and urban mobility platform designed using software engineering methodologies.

## Overview

Felsikka provides:

- Intelligent driver matching
- Real-time GPS tracking
- Fare estimation
- Feedback systems
- Admin analytics dashboard
- Accessibility support for deaf/hard-of-hearing drivers

---

## Features

- Role-based authentication
- Smart ride dispatching
- Live trip tracking
- Surge pricing
- Rating system
- Accessibility-first UI
